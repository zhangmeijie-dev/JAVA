代码块：
	局部代码块------------定义在方法或语句中的代码块 
	
	构造代码块------------定义在成员位置上的代码块
		  ------------优先于构造方法
		  ------------建一次对象就执行一次
	
	静态代码块 ------------也是定义在成员位置上的代码块
		 ------------优先构造代码块运行，    类加载时运行
		 ------------不管建几次对象，都执行一次
		 ------------只能对类中的静态的对象进行操作	
注意：
	1：代码块可以嵌套使用
	2：在嵌套的代码块内，不可以重复定义重名的变量，他并不会向方法那样屏蔽掉其他重名变量
		{
		     int a=3;
	                    {
		        int a=1;-------------------->编译报错      不可以重复定义变量
		    }
	
		}
	3：代码块类定义的变量名不能与所方法或类中的变量重名

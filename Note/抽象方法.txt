★★抽象类：
      1：private 关键字不能和  abstract 关键字不能一起使用 
      2：抽象类不能创建对象（就算抽象类中一个抽象的方法也没有）						
      3：抽象类中可以有非抽象的方法，也可以有非抽象方法
      4：抽象类中的成员变量和非抽象类中的成员变量一样
          （没有抽象变量这一概念，所以成员变量前不可以加上abstract）
      5:   抽象类和一般的类没什么区别只是在抽象类中存在的有抽象的方法，不能创建对象

★★抽象类的定义格式：
      权限修饰符    abstract    class  类名 {
                        ・・・・・・・・・・・
       }

★★抽象方法：
	1：抽象的方法只能存在抽象的类中或者接口中
	2：抽象的方法只能被  protected 或 public、default修饰

★★抽象方法的定义格式：
	权限修饰符   abstract  返回值  方法名 （ 参数列表 ）；
